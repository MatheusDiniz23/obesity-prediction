# Obesity Prediction

Este projeto tem como objetivo prever níveis de obesidade com base em hábitos alimentares e condição física.

## Estrutura do Projeto

- `data/`: Contém o dataset de obesidade.
- `notebooks/`: Notebooks para análise exploratória de dados (EDA).
- `src/`: Scripts de processamento, treinamento e avaliação.
- `app/`: Aplicação para interface de predição.
- `requirements.txt`: Dependências do projeto.
